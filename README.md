# Pandas examples
Демонстрация автоматизации работы с excel файлом

В sample.xlsx находится список задач для выполнения

## Как запустить проект

клонируем:
```
$ git clone https://github.com/lovelydaemon/excel_automatization
$ cd excel_automatization
```
создаем виртуальное окружение, активируем его и устанавливаем необходимые пакеты
из файла requirements.txt:
```
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```

запуск проекта:
```
$ python3 main.py
```
результатом будет новый xlsx
